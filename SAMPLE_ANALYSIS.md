## Methodology Notes

- **Activity Score**: Based on recent commits, contributors, stars, and issue engagement
- **Complexity Score**: Evaluated via language diversity, file count, dependencies, and team size
- **Learning Difficulty**: Combination of activity and complexity scores
- **Maintainability**: Recent commit frequency, issue resolution rate, and team engagement
- **Learning Score**: Optimized for beginner-friendly resources

All scores are normalized to 0-100 scale and calculated using open-source data from GitHub API.
